# Preview Documentation Site

Start the Astro development server for local preview.

## Instructions

1. Check if dependencies are installed
2. Start the dev server on http://localhost:4321
3. Report the URL and any startup errors

## Commands

```bash
# Install dependencies if needed
npm install

# Start dev server
npm run dev
```

## Expected Behavior

- Dev server starts on http://localhost:4321
- Hot module reload enabled for live editing
- Changes to MDX files reflect immediately
